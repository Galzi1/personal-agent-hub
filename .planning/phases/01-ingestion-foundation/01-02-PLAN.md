<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| RSS feed content -> RawItem fields | title, link, summary are untrusted strings from external feeds |
| OpenRouter API response -> verdict parsing | JSON content from OpenRouter is parsed; malformed response must not crash pipeline |
| config/microclaw.config.yaml -> API key | Secrets file read at runtime; key must never appear in logs |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | ingester.py RawItem fields | mitigate | pydantic BaseModel validates all field types at construction; title/link/summary treated as opaque strings - never evaluated or interpreted as code |
| T-02-02 | Tampering | filter.py JSON parse | mitigate | json.loads() wrapped in try/except raises FilterError on malformed content - response content is not evaluated as code |
| T-02-03 | Information Disclosure | load_openrouter_key() | mitigate | API key returned as string and passed as parameter - key never logged or included in FilterError messages |
| T-02-04 | Denial of Service | httpx.Client timeout | mitigate | timeout=30.0 on feed fetch (ingester.py); timeout=90 on OpenRouter call (filter.py) - prevents pipeline hang on slow feeds |
| T-02-05 | Spoofing | User-Agent header | accept | "PersonalAgentHub/0.1" identifies the client truthfully; public RSS feeds require no authentication |
</threat_model>
