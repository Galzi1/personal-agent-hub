<task type="auto" tdd="true">
  <name>Task 1: Setup Ollama and select LLM/Embedding models</name>
  <files>.planning/phases/00-validation-spikes/00-MODEL-EVALUATION-RESULTS.md</files>
  <behavior>
    - Ollama is installed and executable.
    - System RAM is queried and recorded.
    - The appropriate LLM (`qwen3:32b` or `gemma4:27b-it` based on RAM) is pulled.
    - `nomic-embed-text` embedding model is pulled.
    - Model pull commands are recorded.
    - Latency for a sample prompt is measured for the chosen LLM.
  </behavior>
  <action>
    1. Check total system RAM using `systeminfo | findstr "Total Physical Memory"`. Record the value in MODEL-EVALUATION-RESULTS.md.
    2. If RAM >= 32GB, select and pull `qwen3:32b` using `ollama pull qwen3:32b`. Otherwise, select and pull `gemma4:27b-it` (using the instruct-tuned variant for better chat performance) using `ollama pull gemma4:27b-it`. Record the chosen LLM and the pull command.
    3. Pull the embedding model: `ollama pull nomic-embed-text`. Record the command.
    4. Execute a sample prompt (e.g., "What is the capital of France?") with the chosen LLM and record the response and latency in MODEL-EVALUATION-RESULTS.md.
  </action>
  <verify>
    <automated> 
      rtk bash --command "systeminfo | findstr 'Total Physical Memory'" --description "Check system RAM" &&
      rtk bash --command "ollama pull qwen3:32b" --description "Pull Qwen3 model" &&
      rtk bash --command "ollama pull gemma4:27b-it" --description "Pull Gemma4 model" &&
      rtk bash --command "ollama pull nomic-embed-text" --description "Pull embedding model" &&
      rtk bash --command "ollama run qwen3:32b 'What is the capital of France?' --num-predict 20" --description "Test qwen3 latency" &&
      rtk bash --command "ollama run gemma4:27b-it 'What is the capital of France?' --num-predict 20" --description "Test gemma4 latency"
    </automated>
  </verify>
  <done>System RAM is recorded, Ollama is installed, chosen LLM and embedding model are pulled. Basic latency test for chosen LLM is recorded in MODEL-EVALUATION-RESULTS.md.</done>
</task>
